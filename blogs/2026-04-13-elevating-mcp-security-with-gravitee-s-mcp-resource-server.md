As the Model Context Protocol (MCP) rapidly becomes the standard for connecting AI models to data sources, the focus for platform engineers has shifted from simple connectivity to production-grade security. While Gravitee 4.10 laid the groundwork by introducing the MCP Resource Server, the 4.11 release marks a significant milestone: the transition from a functional proof-of-concept to an enterprise-hardened security model.
